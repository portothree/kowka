# kowka

Quick start projects using Gulp, SASS and BEM methodology



# Structure

```
kowka/
│
├── src/                 # Development source
├── dist/                # Build files
├── tasks/               # Gulp tasks
│
├── gulpfile.js
└── package.json
```



# Commands
* `npm i` — install dependencies
* `npm start` — start dev
